============
.lava-report
============

The ``.lava-report`` job iterates through the submitted jobs from
:doc:`lava-test` and collects the artifacts from these jobs via LAVA REST API.


Job Variables
=============

CI_LAVA_INSTANCE
----------------

The base url of the LAVA server.
